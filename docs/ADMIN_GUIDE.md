# Admin Panel User Guide

Welcome to the BalaAstraTech Admin Panel. This system allows you to manage all content on your portfolio website without writing code.

## Getting Started

1.  **Login**: Access the admin URL and log in with your credentials.
2.  **Dashboard**: The home screen shows an overview of your projects, services, and unread messages.

## Managing Content

### Projects (Portfolio)
*   **View**: Click "Projects" in the sidebar to see all portfolio items.
*   **Add**: Click "Add New Project". Fill in the title, description, and upload a cover image.
*   **Edit**: Click the pencil icon on any project.
*   **Tech Stack**: You can select technologies used in the project from the dropdown.

### Services & Capabilities
*   **Services**: Define the core services you offer (e.g., "Full Stack Development").
*   **Additional Capabilities**: Add smaller badges for specific skills or tools.

### Tech Stack
*   **Layers**: Organize technologies into categories (e.g., "Frontend", "Backend").
*   **Items**: Add specific technologies to each layer.

### Trust Signals
*   Manage the logos or statistics that appear in the "Trust" section of your site.

## Configuration

### Site Settings
*   **Hero Section**: Update the main title, subtitle, and hero image.
*   **Contact Info**: Update your email, phone, and social media links.
*   **Page Sections**: Customize the titles and subtitles of every section on the landing page.

### Form Options
*   **System Types**: Manage the options users see in the "Project Type" dropdown on the contact form.
*   **Budget Ranges**: Manage the budget options.

## Inbox
*   View all messages sent through the contact form.
*   Mark messages as "Read" or "Archived" to keep your inbox clean.

## Troubleshooting
*   **Images not showing?** Try refreshing the page. Large images might take a moment to process.
*   **Changes not visible on public site?** Refresh the public site. If utilizing caching, changes might take up to 60 seconds to appear.
